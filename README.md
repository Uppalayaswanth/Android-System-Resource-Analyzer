# Android System Resource Analyzer

Android System Resource Analyzer is an Android application developed as a final year engineering project. The application provides detailed, real-time insights into smartphone hardware, system resources, and performance through a clean and intuitive interface.

---

## Key Features

- CPU and processor information  
- Memory (RAM) usage statistics  
- Storage capacity and availability  
- Battery health, voltage, and temperature  
- Display and screen specifications  
- Operating system and kernel details  
- Performance benchmarking  
- Device hardware information  

---

## Technology Stack

- Java  
- Android SDK  
- Android Studio  
- XML for UI design
